# &gt; 和&gt;&gt;
 &gt;是输出重定向，会覆盖内容，&gt;&gt;是追加，不会覆盖

# 例子
 ls -l 会输出一些内容，这些叫输出，&gt;a.txt会写入a.txt，当然也可以用&gt;&gt;来追加,后面只演示&gt;,不演示&gt;&gt;了
```
ls -l > a.txt
```
<!---more-->

# 例子2
 将cat的输出重定向到b.txt中
```
cat a.txt > b.txt
```

# echo
 输出 abcde
```
echo "abcde"
```
 将abcde写入a.txt
```
echo "abcde" > a.txt
```

# cal
 cal显示日历
 将日历输出到a.txt
```
cal > a.txt 
```
